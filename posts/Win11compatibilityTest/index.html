Windows 11 Compatibility Test

How to check if your PC is capable of running Windows 11

Microsoft Announces Windows 11,but a lot of PC's are incompatible. Here's how you check if your PC is capable of running Windows 11. Windows 11 is Microsoft's brand new operating system that has not yet been released. There are a lot of changes comparing Windows 10 to Windows 11, the UI has changed, the taskbar icons and start button has been centered with the option to revert back to the classic left oriented taskbar, of course. Another major change is that from now, Android apps are coming to Windows. Yes, you heard that right, fully functioning Android apps are coming to Windows via the Microsoft Store. With all of these upsides, there are a lot of downsides of Windows 11. The most major and concerning being that Windows 11 has ditched support for a lot of semi-old processors, mostly the ones made before 2018. Windows 11 Requires you to have Trusted Platform Module 2.0 (aka TPM 2.0) on your computer. The problem is, majority of semi-old PC's do not have TPM 2.0. The below methods are to check if your PC is capable of running this brand new operating system.

WhyNotWin11

WhyNotWin11 is a detailed utility tool that checks your hardware and tells you if your pc is capable of running the upcoming operating system

    • Windows PC Health Check

    The Windows PC Health Check app is an official app from Microsoft that can check if your PC can run Windows 11. However, there are a lot of downsides to this compared to WhyNotWin11, as it does not show detailed information on why/why not your pc is compatible
